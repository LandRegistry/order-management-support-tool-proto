Coded design prototype

What is the purpose of this prototype?

This is a coded version of the Business Gateway Order Management tool designs. These designs are non-functional and are minimally interactive. They are intended to be an additional way for developers to view the designs, allowing for cleaner CSS styles than what Figma can output.

Pages

These are links to the various pages of the tool. Because the pages are non-functional, this is the best way to find the page you are looking for.

Search - Filters open

This shows the search page with filters open, but before the user has taken any action that would create search results.

Search - Filters open, with search results

This shows the search page with the filters open after a user has taken an action that creates search results.

Search - Filters open, no results found

This shows the search page with the filters open and when no results fit the conditions of a search.
